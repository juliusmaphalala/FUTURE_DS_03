# Marketing Funnel and Conversion Analysis for Financial Products

This analysis evaluates marketing funnel performance and conversion rates across customer journeys. Using visualization dashboards and performance metrics,
it highlights optimization opportunities in acquisition and retention stages.

The project models customer acquisition costs, conversion to product usage, and lifetime value in banking or investment contexts.
It includes multi-chart dashboards for stakeholder reporting.
Technologies: Python, Pandas, Matplotlib.
## Relevance to Financial Engineering
- Supports data-driven decisions on campaign ROI.
- Enables segmentation and personalization for financial offerings.
- Builds skills in funnel analytics transferable to credit, insurance, or wealth management.
